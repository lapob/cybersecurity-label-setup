# Cybersecurity Lab - Kali Linux

## Overview
This repository documents my personal cybersecurity lab setup using Kali Linux in a virtual environment.

## Environment
- OS: Kali Linux
- Virtualization: VirtualBox
- Network: NAT

## Setup Steps
- Kali Linux installation in VM
- Automatic disk partitioning
- GRUB installed on /dev/sda
- Display manager: gdm3

## Tools Installed
- nmap
- wireshark
- gobuster

## Skills
- Kali Linux basics
- Virtual machine setup
- Networking basics
- Android Debug Bridge (ADB)
- Firmware analysis
- Root concepts with Magisk

## Notes
This lab is used for cybersecurity learning and testing in controlled environments only.
